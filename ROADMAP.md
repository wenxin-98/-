# 开发路线图

## v1.1 (计划中)

- [ ] **多用户管理**: 用户注册、角色 (admin/user/viewer)、每用户流量配额
- [ ] **流量限额**: 按规则/用户设置流量上限，超限自动停用
- [ ] **Telegram Bot**: 节点离线告警、流量日报、登录通知
- [ ] **数据导入/导出**: 规则批量导入 JSON、数据库备份下载/上传恢复
- [ ] **自动更新检查**: 面板启动时检测 GitHub 最新版本

## v1.2

- [ ] **国际化 (i18n)**: 英文界面
- [ ] **XTLS Reality 自动生成**: 一键生成 x25519 密钥对 + shortId
- [ ] **负载均衡**: 多出口节点轮询/权重分配
- [ ] **Cloudflare WARP 集成**: 一键配置 WARP 出口
- [ ] **落地 IP 检测**: 自动检测各节点出口 IP 和地区

## v1.3

- [ ] **可视化拓扑编辑器**: 拖拽式多跳链路编辑
- [ ] **日志查询**: 全文搜索、时间筛选、导出
- [ ] **Webhook**: 自定义事件回调 (节点状态变更、流量告警)
- [ ] **移动端适配**: 响应式重构 + PWA

## 长期

- [ ] gRPC Agent 通信 (替代 HTTP 心跳)
- [ ] Prometheus metrics 导出
- [ ] 面板集群 (高可用)
